Fort Smash Twitch Stream Overlay

This project is an overlay for the Fort Smash Twitch stream. It was built using an Express JavaScript web server and a MariaDB database to store website data. To get tournament updates from their tournament management website, the Start.gg GraphQL API was used. Lastly, Vue.js was used for the overlay.
